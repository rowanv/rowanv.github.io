Don't Get Kicked Dataset

This dataset was used for the "Don't Get Kicked" competition on Kaggle. The objective of the competition was to predict which cars would be bad buys or "kicks". The training data set consisted of 72983 and 34 variables. The data set is freely available and can be downloaded through Kaggle.

Cars Dataset

This dataset gives the speed of cars and the corresponding distance taken to stop. The data set has 50 observations, and is available within the datasets package in R.

Linear Regression

I fitted the data to a linear model. The linear model yields the equation Y = 3.9324X - 17.5791. For each additional unitary increase in speed, measured in mph, the model predicts an increase of 3.9324 feet before the car will stop. The results are highly statistically significant, with a p value < 0.001.
